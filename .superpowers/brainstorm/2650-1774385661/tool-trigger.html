A — Floating Toolbar on Selection

Current approach. A mini toolbar appears above selected text. Click a tool → results open in the side panel. Familiar, discoverable. But: adds visual noise, covers text on mobile.

B — Persistent Top Toolbar

Tool buttons live in the editor header bar (like Sudowrite). Select text, then click a tool in the toolbar. Always visible, no popups. But: takes toolbar space, less contextual.

C — Hybrid: Top Toolbar + Selection Hint

Tools live in the header. When text is selected, a subtle hint appears near the selection pointing to the toolbar. Best of both: always visible + contextual. Clean, professional, scales to mobile (tools become icons).
